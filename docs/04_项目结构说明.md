# 项目结构说明

## 目录总览

```
Medical_PBL/
├── ai_roles/              # AI 角色规则卡
├── databases/             # 数据库文件（JSON 存储）
├── deploy/                # 部署相关配置
├── docs/                  # 项目文档
├── modules/               # 核心业务模块
│   ├── pbl/               # PBL 基础模块
│   ├── pbl2/              # PBL2 模块（含 DeepSeek 集成）
│   └── pbl3/              # PBL3 模块（含 DeepSeek 集成）
├── public/                # 前端静态资源
├── scripts/               # 工具脚本
├── templates/             # 模板文件
├── server.js              # 主服务入口
├── package.json           # 项目依赖配置
├── .env.example           # 环境变量示例
└── README.md              # 项目说明
```

## 核心目录详解

### `ai_roles/` — AI 角色规则卡

定义 AI 角色的行为规范和触发规则：

| 文件 | 说明 |
|------|------|
| `AI_助教团总规则.md` | 所有 AI 角色的总体行为规范 |
| `AI_触发路由规则.md` | 触发条件与角色选择逻辑 |
| `AI_兜底回复模板.md` | 异常情况下的兜底回复 |
| `A教授_角色技能卡.md` | A教授角色的详细设定 |
| `B同学_角色技能卡.md` | B同学角色的详细设定 |
| `C同学_角色技能卡.md` | C同学角色的详细设定 |

### `modules/` — 核心业务模块

| 文件 | 说明 |
|------|------|
| `server.js` | Express 主服务，路由和中间件 |
| `auth.js` | 教师认证模块 |
| `storage.js` | 数据持久化（JSON 文件存储） |
| `recordService.js` | 课堂记录管理服务 |
| `analyzer.js` | 学生发言分析引擎 |
| `aiReplyRouter.js` | AI 回复路由分发 |
| `aiRoleCards.js` | AI 角色卡加载与管理 |
| `knowledgeBase.js` | 知识库管理 |
| `recommendationEngine.js` | 学习推荐引擎 |
| `docxExporter.js` | Word 文档导出 |
| `safePath.js` | 安全路径校验 |
| `pbl/` | PBL 基础模块 |
| `pbl2/` | PBL2 模块（含 DeepSeek 客户端） |
| `pbl3/` | PBL3 模块（含 DeepSeek 客户端） |

### `public/` — 前端静态资源

| 文件 | 说明 |
|------|------|
| `index.html` | 主页面（SPA 入口） |
| `email.html` | 邮件模板页面 |
| `network-test.html` | 网络连通性测试页面 |
| `` | 静态图片资源 |

### `scripts/` — 工具脚本

| 文件 | 说明 |
|------|------|
| `smoke-test.js` | 冒烟测试脚本 |
| `quality-gate.js` | 质量门禁检查 |
| `benchmark-pbl-multicase.js` | PBL 多病例基准测试 |
| `websocket-ui-path-evidence.js` | WebSocket 路径证据测试 |
| `open-firewall.ps1` | Windows 防火墙开放脚本 |

### `deploy/` — 部署配置

| 路径 | 说明 |
|------|------|
| `deploy/nginx/` | Nginx 反向代理配置 |
| `deploy/systemd/` | Systemd 服务配置 |

### `databases/` — 数据存储

使用 JSON 文件进行轻量级数据持久化：

| 路径 | 说明 |
|------|------|
| `databases/lesson_summaries/` | 课堂总结数据 |
| `databases/.pbl_evidence_index.json` | PBL 证据索引 |

### `templates/` — 模板文件

| 文件 | 说明 |
|------|------|
| `ai_role_review_prompt.md` | AI 角色审查提示模板 |

## 启动文件

| 文件 | 说明 | 平台 |
|------|------|------|
| `server.js` | 主服务入口 | 跨平台 |
| `start.bat` | 一键启动脚本 | Windows |
| `start-server.bat` | 服务器启动脚本 | Windows |
| `start-pbl3.bat` | PBL3 模式启动 | Windows |
| `start-pbl3.ps1` | PBL3 模式启动 | Windows PowerShell |
| `start-and-open.bat` | 启动并打开浏览器 | Windows |
| `start-ui-test.bat` | UI 测试启动 | Windows |
