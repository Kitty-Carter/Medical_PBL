# 更新日志（Changelog）

本文档记录 Medical_PBL 项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.0.0] - 2026-05-03

### 新增
- 初始开源版本发布
- PBL（问题导向学习）病例管理系统核心功能
- 教师端：病例创建、编辑、发布、评分管理
- 学生端：病例查看、答题提交、AI 反馈查看
- DeepSeek AI 集成：自动评分、智能反馈、多角色 AI 助手
- 基于角色的权限管理（教师/学生）
- 文件上传与管理（支持图片、文档等）
- 响应式 Web 界面，支持桌面和移动端
- RESTful API 接口
- 环境变量配置系统
- PM2 进程管理支持
- Nginx 反向代理配置模板
- Docker 容器化部署支持
- 完整的项目文档体系

### 技术栈
- 后端：Node.js + Express.js
- 前端：原生 HTML/CSS/JavaScript + EJS 模板引擎
- 数据库：MySQL
- AI：DeepSeek API（deepseek-chat / deepseek-reasoner）
- 部署：PM2 + Nginx + Ubuntu 22.04 LTS

### 已知问题
- 大文件上传时偶现超时（建议限制 50MB 以内）
- AI 评分在高并发场景下响应延迟增加
- 部分浏览器兼容性问题（建议使用 Chrome/Edge 最新版）
