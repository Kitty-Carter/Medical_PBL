# 快速部署指南

## 方式一：直接部署（推荐用于测试）

### 1. 环境准备

```bash
# Ubuntu/Debian
sudo apt update && sudo apt install -y nodejs npm

# CentOS/RHEL
sudo yum install -y nodejs npm
```

### 2. 安装依赖

```bash
cd Medical_PBL_OpenSource_Release
npm install --production
```

### 3. 配置环境变量

```bash
cp .env.example .env
nano .env  # 编辑配置
```

必须修改的配置项：
- `TEACHER_PASSWORD`：设置教师密码
- `SESSION_SECRET`：设置随机字符串
- `DEEPSEEK_API_KEY` 或 `OPENAI_API_KEY`：配置 AI 服务

### 4. 启动服务

```bash
npm start
```

服务默认运行在 `http://localhost:3000`

## 方式二：Docker 部署（推荐用于生产）

```bash
docker build -t medical-pbl .
docker run -d -p 3000:3000 --env-file .env --name medical-pbl medical-pbl
```

## 方式三：使用部署脚本

```bash
chmod +x deploy/install_ubuntu.sh
sudo bash deploy/install_ubuntu.sh
```

## 验证部署

```bash
curl http://localhost:3000
```
